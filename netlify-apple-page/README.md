# Netlify Apple Page

This folder contains a single-page Privacy + Support site for Moodot.

Deploy this folder directly to Netlify:

1. Open Netlify.
2. Choose "Add new site" > "Deploy manually".
3. Drag the `netlify-apple-page` folder.

Main file:

- `index.html`

If you want to change the support email later, edit the `mailto:` link and visible email text in `index.html`.
